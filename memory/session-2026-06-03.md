## Key decisions made

- **Em-dashes: 100% banned** in spec documents (no exceptions, including dialogue). writing-style.md is the authority.
- **Slop phrases** moved out of skills into `docs/slop-phrases.md` — review reference only, never loaded by agents. Added "gap" as first entry in new AI vocabulary category.
- **"reads as" phrasing** — in slop-phrases.md, not in any skill.
- **Concept + Inspirations → Design Notes** — all builder context is one bullet-point section, excluded from exports.
- **Preamble → `brief` frontmatter** — replaces preamble body section, `role`, and `archetype` (both were ainime artifacts). `brief` is a condensed cast navigation summary.
- **Storylines → story notes** — Storylines section removed from character notes; character arcs become intention notes with `characters: ["[[Name]]"]` frontmatter.
- **Rename script** — `scripts/rename-note.ps1` handles all Obsidian wikilink variants; must be run after any vault file rename.
- **Blueprint rework (#13):** Soul, Behavioral Descriptions, and Relationship Behavior all dissolved into a single behavioral format. New structure: Background (declarative fact pairs) / Body (physical behavioral descriptions) / Soul (psychological + social register behavioral descriptions, absorbing Behavioral Descriptions and Relationship Behavior). Session flow is now a Q&A phase (grill-style: one question at a time, offer hypothesis, follow threads, sharpen vague answers) with answers captured in Design Notes → Session Notes. Relationships entries now use `**Name — Archetype(s):**` bullet format. Intimate Dynamics rewritten to behavioral-only format.
